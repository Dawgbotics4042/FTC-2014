Create a unique ID (UID) for an NXT. This based on the last 3 bytes of the Bluetooth address. The first 3 bytes are manufacturer specific and identical for all NXTs and are therefore not used.

Returns
a unique ID for the NXT.

Enables/Disables the PMW output of the servo controller. This is necessary because of the 10 second timeout of the controller.

Parameters

DaisyChainLevelThe daisy chain level of the controller wanted First controller is 1, not 0.
StatusWhat to set the PMW status to. 0 is on, 170 is on with no timeout, and 255 is off.
