Detailed Description

Code to run hitechnic controllers over I2C.

IMPORTANT:

A. THIS CODE REQUIRES ENCODERS!

B. Standard method argument order is:

void foo(tSensors port, int daisychainLevel, int motorNumber, sbyte bar)

Where foo is your method name and bar is any variable that you need in addition to the standard three. Bar is optional, but your method must always respect motorNumber, port and daisychainLevel.
